# Project Frontend

# Prerequisites
    - Node.js (version 16 or higher)
    - npm package manager

# Installation

Clone the repository and navigate to the directory

Install dependencies

```bash
    npm install
```
# Run the code

Run the development server

```bash
    npm run dev
```

Build for production

```bash
    npm run build
```

Lint code

```bash
    npm run lint
```

Fix linting issues

```bash
    npm run lint:fix
```

Format code with prettier

```bash
    npm run format
```

Check code formating

```bash
    npm run format:check
```

Run tests once
```bash
    npm run test:run
```

Run tests in watch mode
```bash
    npm test
```
